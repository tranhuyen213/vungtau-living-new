# VungTau Living static website

Bộ website tĩnh không dùng npm, không có package.json, phù hợp deploy trực tiếp lên Vercel với Framework Preset: Other.

## Cập nhật lên GitHub

1. Giải nén file zip.
2. Mở repository GitHub đang dùng cho website `vungtau-living-official` hoặc repository hiện tại.
3. Xóa hoặc ghi đè các file cũ bằng toàn bộ file trong thư mục này.
4. Commit changes.
5. Vercel sẽ tự deploy lại nếu repo đã kết nối.

## Cấu hình Vercel

- Framework Preset: Other
- Build Command: để trống
- Output Directory: để trống hoặc nhập `.`
